设置为浏览器主页

如果你觉得本导航对你有用，强烈建议你将本导航设置为浏览器默认打开页面，这样每当你打开浏览器时会自动加载本导航，以方便你实用

各浏览器设置主页方法大同小异，参考如下：

Chrome

打开Chrome浏览器，访问浏览器设置地址：chrome://settings/

找到启动时设置，选择选择特定网页或一组网页，点击添加新网页，填写网址https://nav.chen-hao.com.cn，点击添加即可

设置好后如图所示即表示正常

Microsoft Edge

打开Micorsoft Edge浏览器，访问浏览器设置地址：edge://settings/onStartup

选择打开一个或多个特定页面，点击添加新页面，输入URL网址https://ops-coffee.cn，点击添加即可

添加完成后如果所示即表示正常

360安全卫士绑定

如果你有安装360安全卫士，那么推荐通过360安全卫士的主页防护功能来一键锁定所有浏览器主页，具体方法为

打开360安全卫士，点击顶部功能大全，找到主页防护

点击主页防护弹出设置框，点击锁定浏览器主页为后边的输入框，选择自定义，然后输入https://nav.chen-hao.com.cn，然后点击上方一键锁定

大功告成，祝你使用愉快
